# ☪️ Nūr — Islamic Holistic Self-Growth App

> *"Grow with purpose. Rooted in faith."*

A beautifully designed, production-ready Flutter mobile application for Muslims who want to improve spiritually, mentally, emotionally, physically, and financially — all aligned with the Quran and Sunnah.

---

## 📸 App Overview

**Nūr** (meaning *light* in Arabic) is a holistic self-development companion built for Muslims. It combines the wisdom of Islamic tradition with modern behavioral science and habit-building techniques to help you build a purposeful, growth-oriented life.

### Five Pillars of Growth
| Pillar | Features |
|---|---|
| 🕌 **Spiritual** | Salah tracker, Quran reading log, Dhikr counter, Daily faith score |
| 🧠 **Mental** | Reflection journal, Guided prompts, Growth insights |
| 💛 **Emotional** | Gratitude practice, Mood tracking, Journaling |
| 💪 **Physical** | Exercise habits, Tahajjud tracker, Habit streaks |
| 💰 **Financial** | Sadaqah tracking, Financial habit building |

## 🔥 Firebase Setup

### Step 1: Create Firebase Project

1. Go to [Firebase Console](https://console.firebase.google.com)
2. Click **"Add project"**
3. Name it (e.g., `nur-islamic-growth`)
4. Enable Google Analytics (optional)
5. Click **Create project**

### Step 2: Enable Authentication

1. In Firebase Console → **Authentication** → **Get started**
2. Under **Sign-in method** → Enable **Email/Password**
3. Click **Save**

### Step 3: Set Up Firestore Database

1. In Firebase Console → **Firestore Database** → **Create database**
2. Choose **Start in production mode**
3. Select your region (closest to your users)
4. Click **Enable**

### Step 4: Deploy Security Rules

```bash
# Install Firebase CLI if not already installed
npm install -g firebase-tools

# Login to Firebase
firebase login

# Initialize Firebase in your project
firebase init

# Deploy Firestore rules and indexes
firebase deploy --only firestore
```

Or manually copy `firestore.rules` content into the Firebase Console → Firestore → Rules tab.

### Step 5: Connect Flutter to Firebase

```bash
# In your project root
flutterfire configure
```

This will:
- Detect your Firebase project
- Generate `lib/firebase_options.dart` automatically
- Download `google-services.json` (Android) and `GoogleService-Info.plist` (iOS)

> ⚠️ **Never commit** `firebase_options.dart`, `google-services.json`, or `GoogleService-Info.plist` to public repositories. They are listed in `.gitignore`.

### Step 6: Add Fonts (Optional)

Download the **Amiri** font (for Arabic text):
1. Visit [Google Fonts - Amiri](https://fonts.google.com/specimen/Amiri)
2. Download **Amiri Regular** and **Amiri Bold**
3. Place files in `assets/fonts/`

```
assets/fonts/
├── Amiri-Regular.ttf
└── Amiri-Bold.ttf
```

## 📱 Features In Detail

### 1. User Authentication
- Email/password registration and login
- Account creation automatically seeds default Islamic habits
- Secure Firebase Authentication backend
- Password reset flow

### 2. Daily Faith Dashboard
- **Salah Tracker**: Tap each of the 5 prayers to mark as complete
- **Daily Faith Score**: Auto-calculated 0–100 based on spiritual activities
- **Quran Pages**: Quick-select buttons (1, 2, 3, 5, 10 pages)
- **Dhikr Counter**: Tap counter with haptic feedback, tracks Subhanallah/Alhamdulillah/Allahu Akbar
- **Gratitude Note**: Inline editor to note daily thankfulness

### 3. Habit Tracker
- Default Islamic habits auto-created on registration
- Custom habits with 5 categories: Spiritual, Physical, Mental, Emotional, Financial
- 16 custom emojis for habit icons
- 🔥 Streak tracking per habit
- Swipe-to-delete habits
- Today's progress ring with completion count

### 4. Reflection Journal
- Daily journal entries with mood tagging
- 6 mood options: Grateful, Peaceful, Content, Reflective, Struggling, Hopeful
- Guided prompts toggle (10 Islamic prompts)
- New prompt randomizer
- Full entry history

### 5. Growth Dashboard
- Best streak across all habits
- Total active habits count
- Total journal reflections
- Average daily faith score
- Habit completion by category
- Today's full habit checklist
